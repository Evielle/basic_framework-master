# M3508电机控制与编码器角度分析

## 概述

框架提供了完整的DJI智能电机（M3508/M2006/GM6020）控制模块，位于 `modules/motor/DJImotor/`。同时提供了舵机控制模块 `modules/motor/servo_motor/`。

**本文重点回答三个问题：**
1. 如何获取M3508编码器角度
2. 如何给M3508发送低速转动指令
3. 舵机/校准相关功能

---

## 一、M3508编码器角度获取

### 1.1 编码器数据结构

位置：`modules/motor/DJImotor/dji_motor.h` - `DJI_Motor_Measure_s`

```c
typedef struct {
    uint16_t last_ecd;          // 上一次读取的编码器值
    uint16_t ecd;               // 0-8191, 当前编码器值 (刻度0-8191)
    float angle_single_round;   // 单圈角度 (0~360度)  <- 【重点】
    float speed_aps;            // 角速度 (度/秒)
    int16_t real_current;       // 实际电流
    uint8_t temperature;        // 温度 (摄氏度)
    float total_angle;          // 总角度 (多圈, 带正负)  <- 【重点】
    int32_t total_round;        // 总圈数
} DJI_Motor_Measure_s;
```

### 1.2 访问方式

```c
DJIMotorInstance *my_motor;  // 假设已经初始化

// 方式1：单圈绝对角度 (0~360度)
float single_angle = my_motor->measure.angle_single_round;

// 方式2：多圈累计角度 (可正可负, 用于位置闭环)
float total_angle = my_motor->measure.total_angle;

// 方式3：原始编码器值 (0~8191)
uint16_t raw_ecd = my_motor->measure.ecd;

// 方式4：角速度 (度/秒)
float speed = my_motor->measure.speed_aps;

// 方式5：总圈数
int32_t rounds = my_motor->measure.total_round;
```

### 1.3 转换公式

框架内部使用的转换系数：
```c
#define ECD_ANGLE_COEF_DJI 0.043945f  // 360 / 8192
```

| 数据 | 公式 | 范围 |
|------|------|------|
| 单圈角度 | `angle_single_round = ECD_ANGLE_COEF_DJI * ecd` | 0~360 |
| 角速度 | 从RPM转换, 带低通滤波 | 正负 |
| 总角度 | `total_round * 360 + angle_single_round` | 任意值 |

### 1.4 多圈角度计算原理

在 `DecodeDJIMotor()` 中实现：
```c
// 假设两次采样间电机转过的角度小于180°
if (measure->ecd - measure->last_ecd > 4096)     // 从8191跳转到0附近(正向)
    measure->total_round--;
else if (measure->ecd - measure->last_ecd < -4096) // 从0跳转到8191附近(反向)
    measure->total_round++;
measure->total_angle = measure->total_round * 360 + measure->angle_single_round;
```

**关键假设**：两次采样间电机转过的角度不超过180°。正常运行下（控制频率>500Hz）成立。

## 二、M3508发送低速转动指令

### 2.1 方式一：速度环闭环控制（推荐）

**Step 1: 初始化电机（设置为速度+电流双环）**

```c
Motor_Init_Config_s config = {
    .motor_type = M3508,
    .can_init_config = {
        .can_handle = &hcan1,   // CAN1或CAN2
        .tx_id = 1,             // 电机ID (电调闪动次数, 1-8)
    },
    .controller_setting_init_config = {
        .angle_feedback_source = MOTOR_FEED,
        .speed_feedback_source = MOTOR_FEED,
        .outer_loop_type = SPEED_LOOP,    // 最外层是速度环
        .close_loop_type = SPEED_LOOP | CURRENT_LOOP,  // 启用速度环+电流环
        .motor_reverse_flag = MOTOR_DIRECTION_NORMAL,  // 方向正常
    },
    .controller_param_init_config = {
        // 电流环PID参数
        .current_PID = {
            .Kp = 1, .Ki = 0, .Kd = 0,
            .MaxOut = 16000,     // M3508最大电流输出(-16384~16384)
            .Improve = PID_Trapezoid_Intergral | PID_Integral_Limit | PID_Derivative_On_Measurement,
            .IntegralLimit = 5000,
        },
        // 速度环PID参数 (低速时需要调整)
        .speed_PID = {
            .Kp = 20,            // 比例系数,低速时适当减小
            .Ki = 0.5,           // 积分系数,低速时增加
            .Kd = 0.1,           // 微分系数,避免超调
            .MaxOut = 10000,     // 输出限幅(传递给电流环)
            .Improve = PID_Trapezoid_Intergral | PID_Integral_Limit | PID_Derivative_On_Measurement,
            .IntegralLimit = 5000,
            .Output_LPF_RC = 0.3,
        },
    },
};

DJIMotorInstance *my_motor = DJIMotorInit(&config);
```

**Step 2: 设置低速速度参考**

```c
// 设定值单位: 度/秒
// M3508空载最大转速约480rpm = 2880度/秒
// 低速建议: 100~500度/秒 (约17~83rpm)

DJIMotorSetRef(my_motor, 300.0f);  // 以300度/秒的速度转动
```

**Step 3: 确保电机控制任务在运行**

位置：`modules/motor/motor_task.c` - `MotorControlTask()`

```c
// 该函数已经在 robot_task.h 的 StartMOTORTASK() 中被调用
// 执行频率: 500Hz (通过osDelay(2)控制)
MotorControlTask() {
    DJIMotorControl();   // <-- DJI电机控制函数
    PowerControl();      // <-- 功率控制版
    // ...
}
```

### 2.2 方式二：开环电流控制（不推荐，仅特殊场景）

```c
// 初始化时配置为只有电流环
Motor_Init_Config_s config = {
    .motor_type = M3508,
    .can_init_config = {.can_handle = &hcan1, .tx_id = 1},
    .controller_setting_init_config = {
        .outer_loop_type = CURRENT_LOOP,
        .close_loop_type = CURRENT_LOOP,  // 仅电流环
        // ...
    },
    .controller_param_init_config = {
        .current_PID = {.Kp = 1, .Ki = 0, .Kd = 0, .MaxOut = 16000},
    },
};

// 设定值为电流 (A), 对应-16384~16384
// 小电流(低速)示例: 给1~5A,对应819~4096
DJIMotorSetRef(my_motor, 2.0f);  // 2A电流(低速)
```

### 2.3 方式三：位置环控制（精确角度定位）

```c
// 初始化配置: 位置环 + 速度环 + 电流环
Motor_Init_Config_s config = {
    .motor_type = M3508,
    // ...
    .controller_setting_init_config = {
        .outer_loop_type = ANGLE_LOOP,     // 最外层是位置环
        .close_loop_type = ANGLE_LOOP | SPEED_LOOP | CURRENT_LOOP,  // 三环
        // ...
    },
    .controller_param_init_config = {
        .angle_PID = {
            .Kp = 5,            // 位置环比例
            .Ki = 0,
            .Kd = 0.1,
            .MaxOut = 2000,     // 输出为速度参考(度/秒)
            .IntegralLimit = 1000,
            .Improve = PID_Trapezoid_Intergral | PID_Integral_Limit | PID_Derivative_On_Measurement,
        },
        .speed_PID = { /* ... */ },
        .current_PID = { /* ... */ },
    },
};

// 设置位置参考(度)
// 低速转动: 缓慢从当前角度移动到目标角度
float target_angle = my_motor->measure.total_angle + 360.0f;  // 转1圈
DJIMotorSetRef(my_motor, target_angle);
```

### 2.4 低速控制技巧

1. **适当减小位置环Kp**: 避免低速时出现抖动
2. **增加积分项作用**: 低速摩擦力占比大,需要积分补偿
3. **使用死区**: 小误差时不进行控制
4. **控制频率**: 确保 `MotorControlTask()` 执行频率≥500Hz

### 2.5 检查电机状态

```c
// 读取当前速度
float current_speed = my_motor->measure.speed_aps;

// 读取当前电流
int16_t current = my_motor->measure.real_current;

// 读取温度
uint8_t temp = my_motor->measure.temperature;

// 电机使能/失能
DJIMotorEnable(my_motor);  // 使能
DJIMotorStop(my_motor);    // 失能(停止响应参考输入)
```

## 三、舵机控制

### 3.1 舵机类型

位置：`modules/motor/servo_motor/servo_motor.h`

```c
typedef enum {
    Servo_None_Type = 0,
    Bus_Servo = 1,      // 总线舵机 (UART控制,如华舵舵机)
    PWM_Servo = 2,      // PWM舵机 (普通50Hz PWM舵机)
} ServoType_e;
```

### 3.2 PWM舵机初始化与使用

```c
#include "servo_motor.h"

ServoInstance *my_servo;

// 初始化
Servo_Init_Config_s servo_config = {
    .servo_type = PWM_Servo,
    .pwm_init_config = {
        .htim = &htim3,           // 定时器句柄
        .channel = TIM_CHANNEL_1, // 通道
        .period = 2000,           // PWM周期 (50Hz -> 20ms)
        .duty_ratio = 100,        // 初始占空比
    },
    .servo_id = 0,
};

my_servo = ServoInit(&servo_config);

// 设置角度 (PWM舵机: angle参数为占空比,而非角度!)
// 注意: @todo 这里的angle参数实际传给PWMSetDutyRatio作为占空比
// 需要根据舵机型号计算: 180度舵机一般 500us(-90) ~ 2500us(+90)
ServoSetAngle(my_servo, 150);  // 占空比150
```

### 3.3 总线舵机（华舵舵机协议）

```c
Servo_Init_Config_s servo_config = {
    .servo_type = Bus_Servo,
    ._handle = &huart2,          // 串口句柄
    .servo_id = 1,
};

my_servo = ServoInit(&servo_config);

// 设置角度: 发送0x55 0x55协议帧
// @todo 协议实现不完整, angle参数被转成uint16_t后填入第9/10字节
// 推荐范围: 0~1000 (对应0~300度, 取决于舵机型号)
ServoSetAngle(my_servo, 500);  // 转到中间位置
```

**M3508没有内置的自动校准**。编码器是12位绝对值编码器（0~8191），上电时的位置取决于电机静止时的机械位置。

### 4.2 舵机的校准

总线舵机自身通常具有位置记忆功能，上电即可使用。

PWM舵机需要通过实验确定占空比与角度的映射关系：
- 一般: 1ms = -90度, 1.5ms = 0度, 2ms = +90度
- 具体数值需根据舵机型号调整

## 六、如何让电机功能起作用 - 完整步骤

### Step 1: 硬件连接

- **M3508**: CAN1或CAN2, 电调ID通过拨码开关设置(1~8)
- **PWM舵机**: 定时器PWM输出
- **总线舵机**: 串口(UART)

### Step 2: CubeMX配置

确保CAN、定时器PWM、串口外设已正确配置。

### Step 3: 代码中的初始化

```c
// 在应用层初始化中 (如RobotInit)
// 1. 创建电机实例并保存指针
DJIMotorInstance *my_motor;
Motor_Init_Config_s motor_config = { /* 配置 */ };
my_motor = DJIMotorInit(&motor_config);

// 2. 或使用功率控制初始化(底盘专用)
my_motor = PowerControlInit(&motor_config);

// 3. 确保OSTaskInit()已被调用 (创建电机控制任务)
// 该函数由RobotInit()调用
```

### Step 4: 运行时控制

```c
// 在应用任务中 (如RobotTask)
// 1. 设置参考值 (速度环: 度/秒, 位置环: 度, 电流环: A)
DJIMotorSetRef(my_motor, 500.0f);  // 500度/秒低速

// 2. 读取反馈数据
float speed = my_motor->measure.speed_aps;
float angle = my_motor->measure.total_angle;

// 3. 停止
DJIMotorStop(my_motor);
```

## 七、常见问题排查

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 电机无响应 | CAN ID冲突 | 检查电机ID配置, `tx_id` 是否正确 |
| 电机抖动 | PID参数不合理 | 减小Kp, 增加Kd, 检查DeadBand |
| 反馈为0 | CAN通信失败 | 检查CAN收发器供电、波特率匹配 |
| 速度不准 | 方向反转 | 调整 `motor_reverse_flag` |
| 低速不稳 | 仅用速度环 | 增加位置环做外环 |
| 电流限制 | 功率限制生效 | 检查 `SetPowerLimit()` 设置 |

## 八、关键注意事项

1. **M3508使用C620电调**: 最大电流±20A, CAN ID为电调闪烁次数(1~8)
2. **ID冲突检测**: 同一CAN总线上两个电机的ID不能冲突（自动检测并死循环报错）
3. **反馈单位**: 速度为**度/秒**, 不是rpm (rpm×6 = 度/秒)
4. **位置单位**: 角度为**度制**, 不是弧度
5. **最大电机数**: 每个CAN总线不超过6个DJI电机, 总电机数不超过12个
6. **舵机协议**: `servo_motor.c` 中的总线舵机协议实现不完整（@todo标识），可能需要修改
7. **PWM舵机角度换算**: `ServoSetAngle()` 中PWM舵机的angle参数实际是占空比，不是角度值，需自行换算
8. **控制频率**: `DJIMotorControl()` 必须在实时任务中以固定频率(≥500Hz)调用
